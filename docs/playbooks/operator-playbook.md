## 📱 Manager (Phone-First)
**Mission**: Roster compliance and Act assembly.

1. **Morning Prep**: Open `/dashboard` on mobile. Sync Roster from source.
2. **Compliance Check**: Filter Attendees by "Identity Not Verified" or "Missing Waiver".
3. **Act Assembly**:
   - Go to `/acts`.
   - Ensure all performers are assigned to their respective Acts.
   - Open **Intro Studio** for each act.
   - Select 2+ photos -> Curate -> Generate Assets -> **Approve**.
4. **Final Check**: Verify "Intro Ready" and "Stage Ready" badges are green for everyone in the current block.

##  tablet-first operator tool (Tablet-First)
**Mission**: Orchestrate the live show.

1. **Console Prep**: Open Stage Console. Select "Main Stage".
2. **Verify Lineup**: Check the "Now -> Next" stack. Ensure backgrounds/introductions are loaded.
3. **Execution**:
   - **Trigger Play Intro** when the MC introduces the act.
   - Observe cinematic playback.
   - **Advance Act** once the performance concludes to update the "Now" slot.
4. **Crisis Handling**: If an intro fails to load, use the "Skip Intro"/manual skip logic to keep show pace.
